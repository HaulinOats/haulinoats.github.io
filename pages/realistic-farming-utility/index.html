Stardew Valley - Realistic Farming Utility

Are you tired of the farming aspect of Stardew Valley being too predictable, knowing you can just resort to posts and articles online that tell you the objectively best crops to maximize profit? This utility will pseudo-randomize all the properties of both seeds and crops to help keep farming balanced but also, randomize crops and seeds values so you'll have to figure out the most efficient crops on your own.

This utility it built to work with the Stardew Valley mod Balanced Valley - Realistic Farming courtesy of o0MrPro.

To view calculations and data output, open the developer console (right click anywhere on the page and select "inspect" then click on "console" panel). You can also reference this link if you're having trouble figuring it out.

How to use

Upload the content.json file that exists inside the [CP] Balanced Valley - Realistic Farming mod folder and then click the Generate content.json file button. The file will be updated with new psuedo-randomized crop values (harvest time, sell prices, regrowth capability, etc). Simply place this file back into the [CP] Balanced Valley - Realistic Farming mod folder, overwriting the existing content.json file. ** Please ensure you place it in the [CP] Balanced Valley - Realistic Farming mod folder and NOT the [STF] Balanced Valley - Realistic Farming mod folder (which handles store-related modifications) **
